# Paglinks
Link: https://thiagobatista5.github.io/Pagina-de-Links/

Uma página pessoal simples e elegante para apresentar meus projetos favoritos e links das minhas redes sociais.

## Descrição

O Paglinks é uma landing page pessoal criada para Thiago Batista, onde ele compartilha seus jogos preferidos e convida visitantes a se conectarem através de suas redes sociais. A página é responsiva e utiliza animações suaves para uma experiência visual agradável.

## Funcionalidades

- **Perfil**: Exibe uma foto de perfil com efeito de hover.
- **projetos Favoritos**: Lista de projetos preferidos com links para das páginas.

- **Redes Sociais**: Links diretos para Twitch, Instagram e LinkedIn.
- **GitHub**: Seção dedicada para acompanhar projetos no GitHub (atualmente em desenvolvimento).
- **Design Responsivo**: Adaptável a diferentes tamanhos de tela.
- **Animações**: Efeitos de entrada e hover para interatividade.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página.
- **CSS3**: Estilização, incluindo gradientes, animações e responsividade.

## Como Usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. Navegue pelas seções para ver os jogos e links sociais.

Não há dependências externas além de uma conexão com a internet para carregar as fontes e ícones.
